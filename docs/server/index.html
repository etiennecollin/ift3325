A simple asynchronous TCP server that listens for client connections and handles incoming data.

This server uses the tokio asynchronous runtime and the log crate for logging. Clients can connect to the server, send data, and if they send the message â€œshutdownâ€, the server will shut down.

Â§Usage

To run the server, specify a port number as a command-line argument:

cargo run -- <port_number> <prob_frame_drop> <prob_bit_flip>

Replace <port_number> with the desired port number for the server, <prob_frame_drop> with the probability of dropping a frame, and <prob_bit_flip> with the probability of flipping a bit in a frame.

The probabilities are given as floating point numbers in the range [0, 1] and are independent.
